CSS Positioning

A blog about CSS positioning properties and how they can be used. Explores static, relative, absolute, and fixed positioning, as well as the properties that are commonly shared between them, such as the z-index and manual positioning properties. I hope to update this blog in the future with explanations of how these positioning properties interact with popular layout modules such as Flexbox and Grid, as there are some unintuitive cases with these that I would like to explore.
